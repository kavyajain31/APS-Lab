## 📘 APS-Lab

A collection of **Algorithm Practice Lab notebooks** created for coursework and self-study. The repository contains multiple Jupyter Notebooks, each focusing
on different algorithmic concepts and exercises.

### 📂 Repository Structure

| `Lab_1_AlgorithmsBasics.ipynb` | Introduction to algorithms, basic examples |
| `Lab_2_AlgorithmsBasics.ipynb` | Continuation of algorithm fundamentals |
| `lab_4 (1).ipynb` | Exercises on recursion and problem-solving |
| `Lab_3_Growth_of_function (1).ipynb` | Growth of functions, asymptotic analysis |
| `Lab_5 (1).ipynb` | Algorithm design exercises |
| `Lab_6 (1).ipynb` | Sorting and searching problems |
| `Lab_7 (1).ipynb` | Divide and conquer techniques |
| `Lab_8 (1).ipynb` | Greedy algorithms |
| `Lab_9 (1).ipynb` | Dynamic programming problems |
| `Lab_10 (1).ipynb` | Advanced DP and optimization |
| `Lab_11 (1).ipynb` | Graph algorithms |
| `Lab_12 (1).ipynb` | Final lab exercises |

### 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/kavyajain31/APS-Lab.git
   ```
2. Navigate into the folder:
   ```bash
   cd APS-Lab
   ```
3. Open notebooks in Jupyter:
   ```bash
   jupyter notebook
   ```

### 🧑‍💻 Requirements
- Python 3.x  
- Jupyter Notebook  
- Common libraries: `numpy`, `matplotlib` (depending on exercises)

### 🎯 Purpose
This repository is designed to:
- Practice algorithm design and analysis.  
- Explore different paradigms: **divide and conquer, greedy, dynamic programming, graph algorithms**.  
- Provide hands-on coding exercises for students learning **Algorithm and Problem Solving (APS)**.
